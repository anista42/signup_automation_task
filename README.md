# Signup Automation Task

## Project Description
This project contains an automation script for testing the signup functionality.

## Tools Used
- Python
- Selenium
- Chrome Browser

## How to Run the Script
1. Install Python on your system
2. Install Selenium using:
   pip install selenium
3. Run the automation script:
   python signup_automation_task.py
